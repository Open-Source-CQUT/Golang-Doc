                                      # 后记

                                      如果你真的将语法篇和进阶篇都仔细阅读完了，并且动手敲了代码，那么你会认识到Go的语法其实并不难，特性也并不多，经验丰富的人可能几天就上手了，它完完全全是一门面向工程的语言，这一点你会在以后的编码中慢慢体会到。倘若你学习完后感到无处施展拳脚，想要更深一步学习，我可以给出下面几个学习路线和示例项目以供参考。

                                      # Web开发

                                      Web开发是Go语言中一个很常见的应用场景，你可以尝试以下的几个开源Web框架：

                                      • Ginopen in new window
                                      • Echoopen in new window
                                      • Irisopen in new window
                                      • Hertzopen in new window
                                      • net/http， 或者不用框架，用Go自带的原生HTTP标准库

                                      如果你想要进行微服务开发，我推荐下面几个开源框架：

                                      • GoZeroopen in new window
                                      • go-kratosopen in new window
                                      • kitexopen in new window

                                      然后你需要与数据库交互，我推荐下面几个开源库：

                                      • sqlxopen in new window
                                      • Gormopen in new window
                                      • Xormopen in new window
                                      • Entopen in new window

                                      你可以参照下面这些项目来进行学习：

                                      • gin vue adminopen in new window
                                      • go-zero-looklookopen in new window
                                      • gin-adminopen in new window，单体Web项目我比较推荐参考该项目进行学习
                                      • go-platformopen in new window，微服务项目我推荐参考该项目进行学习

                                      # 命令行工具

                                      Go非常适合用于命令行CLI工具开发，下面推荐几个命令行开发框架或库：

                                      • flagopen in new window，flag标准库
                                      • cobraopen in new window，强烈推荐使用cobra进行CLI开发，有许多著名项目都是基于cobra，比如Docker，K8S等
                                      • go-prettyopen in new window，用于在命令行进行美观的输出

                                      由于cobra十分常用，基于它进行命令行开发的项目非常多，随便都可以找到，下面只是一些参考项目：

                                      • cobra-exampleopen in new window
                                      • govmopen in new window
                                      • hugoopen in new window
                                      • delveopen in new window

                                      上面只是列出了比较常见的领域，你可以看到的是Go在服务端领域应用较为广泛，而客户端则较少，其实Go在其他领域也有不少的应用，比如webrtc，区块链，对象存储，云计算等等，感兴趣可以在Github上自行了解。
